Application State Machine
DISCOVERED → SCORING → DRAFTING → PENDING_APPROVAL → APPROVED → SUBMITTED
(any state) → FAILED → RETRY → (resumes from failed point)

Enforcement rule: The /applications/{id}/submit endpoint checks status = APPROVED in the database. If not, it returns HTTP 403. This check is in the API handler — it cannot be bypassed by a direct API call.

Phase 2: RAG & Matching Core   ·   Weeks 4–6 · 2 BE + 1 ML

Job scraper workers (pluggable strategy per source, rate-limited, raw HTML stored in S3)

Job embedding pipeline: text-embedding-3-small → pgvector (HNSW index)
✓
Fit Scorer agent: structured JSON output { score, match_pct, rationale, gap_skills, strength_skills }
✓
Pydantic validation on all agent outputs — invalid → FAILED state + alert
✓
Job catalog REST API with filter/sort/pagination
✓
Match feed endpoint: cosine similarity ranking against user résumé embedding
✓
Deduplication: hash(source_url) as unique key per job
🚦 Gate: Scorer reliably produces structured JSON. Invalid outputs are caught and flagged before reaching the DB.

Phase 3: Agentic Layer   ·   Weeks 7–9 · 2 BE + 1 ML
✓
Drafter Agent: email + LinkedIn message drafts stored as status=DRAFT in outreach_drafts
✓
Interview Prep Agent: RAG-grounded with job description + résumé context
✓
Sender Agent: triggered ONLY when applications.status = APPROVED — dispatches email or LinkedIn
✓
LLM interaction logging: agent_name, prompt_hash, raw_output, latency_ms, user_edit, feedback_score
✓
Retry / FAILED state handling with exponential backoff (max 3 retries)
✓

✓

🚦 Gate: Full DISCOVERED → PENDING_APPROVAL flow works end-to-end. Sender cannot fire without APPROVED in DB.

Definition of Done (MVP)
	•	User can register, upload résumé, and complete Career Profile
	•	Job Discovery shows personalized match feed with fit scores
	•	User can track applications through the full pipeline
	•	AI generates outreach drafts and surfaces them in the Approval Dashboard
	•	No draft can be sent without explicit approval (verified by penetration test)
	•	Career Copilot answers career questions with context from user's résumé
	•	ATS Optimizer returns keyword match and improvement suggestions
	•	Stripe subscription flow works end-to-end (Standard / Pro / Ultimate)
	•	All P0 pages pass WCAG 2.1 AA audit
	•	p95 API response < 300ms under 500 concurrent users
	•	GDPR deletion endpoint tested and verified
